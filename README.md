## 💡 Why Hayaa?
Quitting an addiction is incredibly hard. Most traditional blockers fail because they don't rewire the brain. **Hayaa** focuses on *Neuroplasticity* and *Habit Loops*. It provides instant tools to fight sudden urges and tracks daily progress without guilt or shame.

## ✨ Key Features

### 🛡️ Instant Urge Protection (Psychological Tools)
* **Urge Surfing:** A guided step-by-step process to ride out the craving wave.
* **CBT Reframing (AI-Powered):** Cognitive Behavioral Therapy tool using AI to reframe negative/triggering thoughts.
* **4-7-8 Breathing:** A visual breathing exercise to calm the parasympathetic nervous system.
* **5-4-3-2-1 Grounding:** A mindfulness technique to bring the user back to the present moment.
* **5-Minute Timer:** Research shows urges drop significantly after 3-7 minutes. 

### ☪️ Islamic Mode (Optional)
* **Dynamic Prayer Times:** Uses Geolocation and Aladhan API to show accurate local prayer times.
* **Quranic Reminders:** Contextual verses and Hadiths based on the user's daily mood.
* **Digital Tasbih:** A counter for instant Dhikr during moments of weakness.
* **Islamic Habit Tracking:** Points for Salah, Quran recitation, and avoiding the haram.

### 📊 Progress & Analytics
* **Streak & Recovery Timeline:** Tracks clean days alongside a neuroscience-backed timeline (e.g., Dopamine stabilization, hormone balance).
* **Guilt-Free Relapse Logging:** Allows users to log a relapse to find the "Trigger" without feeling ashamed.
* **Daily Habit Loop:** Replaces old bad routines with positive daily actions (Exercise, Journaling, Reading).

## 🛠️ Tech Stack
This app is built to be extremely lightweight, fast, and 100% serverless for maximum privacy.
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+).
* **Storage:** Browser `localStorage` (No user data is ever sent to a database).
* **APIs Used:** * *Aladhan API* (for Prayer Times & Hijri Calendar)
  * *Anthropic API / Proxy* (for CBT thought reframing)
* **Architecture:** Modular file structure (separated CSS, JS states, tools, and UI rendering).
